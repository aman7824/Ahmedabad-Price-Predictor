# Ahmedabad-Price-Predictor
A machine learning-based web application that predicts house prices in Ahmedabad based on various input features like location, area (sq ft), number of bedrooms (BHK), bathrooms, and more.
<br>
📌 Features
📍 Area-wise price prediction (Ahmedabad specific)<br>
🔘 User-friendly UI with dropdown inputs<br>
📊 Data visualization & analytics (charts)<br>
🤖 Advanced ML model (XGBoost / Linear Regression)<br>
⚡ Fast prediction using Flask API<br>
🌐 Fully functional frontend + backend integration<br>
<br><br><br>
🛠️ Tech Stack<br>
Frontend: HTML, CSS, JavaScript<br>
Backend: Python (Flask)<br>
Machine Learning: Scikit-learn / XGBoost<br>
Data Handling: Pandas, NumPy<br>
Visualization: Matplotlib / Chart.js<br><br><br><br>
📂 Project Workflow<br>
Data Cleaning & Preprocessing<br>
Exploratory Data Analysis (EDA)<br>
Feature Engineering<br>
Model Training & Evaluation<br>
Flask API Development<br>
Frontend Integration<br>
<br><br><br>
🎯 Use Case<br>

This project helps users, buyers, and real estate enthusiasts to estimate property prices in Ahmedabad based on real data.
